# PSSA ELA PDF / Source Packet for Claude

This packet is a link inventory of official Pennsylvania Department of Education / DRC sources for PSSA ELA content and format analysis.

Important use note: use these released/training materials to understand item formats, eligible-content alignment, scoring, and interaction surfaces. Do **not** copy passages, stems, answer choices, or screenshots into product content. Several PDE samplers state that permission to copy/use the materials does not extend to commercial purposes.

Also send Claude the DRC INSIGHT screenshots separately, because the static PDFs flatten interactive TEI widgets and do not fully expose the live response surfaces.

## What Claude should extract

- Item-type inventory: MC, EBSR, multi-select, inline drop-down, matching/grid, hot-text, drag-and-drop, short answer, TDA.
- For each item type: schema fields, correct-response encoding, partial-credit rules, evidence-span requirements, student preview, reviewer preview, and audit gates.
- EC skill families from Assessment Anchors: Literature A, Informational B, Key Ideas, Craft/Structure/Integration, Vocabulary, Conventions D, TDA E.
- Do not rely only on PDFs for TEI surfaces; the live DRC screenshots are primary evidence for interaction behavior.
