Design notes · Sýnesis brand architecture

  • Three-layer brand hierarchy. Sýnesis (parent platform · adult-facing gravitas) → Subject Buddy / Planet name (warm kid-facing program) → Test Prep modules (state-specific, span subjects). The visual identity at each layer is distinct: deep indigo + Cormorant Garamond serif for Sýnesis, planet color gradient + Inter sans for each Buddy, slate-900 for Test Prep.
  • Cormorant Garamond for the wordmark. Classical serif echoes the Greek root. Subtle pedigree signal for educators and investors. Could be swapped to Playfair Display or Cardo if procurement preference differs.
  • Planet identities are real visual systems, not just labels. Venus = warm amber-orange (literacy is warm, evocative). Mercury = cool silver-gray (math is precise, structured). Mars = red-orange (science is exploratory). Earth = blue-green (social studies is grounded). When a kid is "on Venus," the chrome shifts subtly amber. Personality without changing the underlying app.
  • Program switcher in the top nav is the source of truth. Kids switch planets explicitly. Sub-nav within each program is program-specific (Reading Buddy has Speed drill; Math Buddy will have Number talks; etc.).
  • Test Prep sits at the Sýnesis level, not inside a program. Reason: tests span subjects. STAAR has math AND reading. Putting Test Prep inside Reading Buddy would mean duplicating it inside Math Buddy. Keeping it at the parent level keeps the architecture clean.
  • "Coming soon" + "Request your state" in the Test Prep dropdown is free product research. You'll know which states have demand from existing customers before you commit engineering.
  • Migration banner uses planet language without explaining it. "Reading Buddy · Venus" reads naturally; the kid or parent figures out the metaphor in context. Curiosity drives the first click.
  • Mercury tab on the parent dashboard shows "not enrolled." Soft, contextual upsell — visible but not pushy. The family can ignore it or add Math Buddy when they're ready.
  • Codex implementation impact (vs. the spec we wrote): brand-name strings change from "Reading Buddy" parent-level to "Sýnesis" parent-level, with "Reading Buddy" demoted to one program label. The program switcher is a new top-level nav component. Each program needs a `colorTheme` config that drives chrome accent colors. Test Prep dropdown component. Schema stays unchanged.
