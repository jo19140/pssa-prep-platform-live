# Pilot UI Mockups

These HTML files are frozen visual references for the Sýnesis Learning pilot UI polish work.

They were restored from repository history in Phase 0 of `specs/pilot-ui-polish-codex-spec.md` and moved under `docs/mockups/` so they remain available for design comparison without being part of the application build path.

Phase 2 implementation PRs should use these mockups as the source of truth for visual polish. Components should match the relevant mockup while preserving existing application data flow, routing, and behavior.
